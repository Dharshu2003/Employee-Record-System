What is ERS?
Employee Record System is software built to handle the primary housekeeping functions of a company. ERS helps companies keep track of all the employees and their records.This software built to handle the records of employees of any company. OBJECTIVES OF THIS PROJECT:
This project aims to simplify the task of maintaining records of the employees of Company.
To develop an well-designed database to store employee information.
Provides full functional reports to management of Company.
The objective of this project is to provide a comprehensive approach towards the management of employee information.
